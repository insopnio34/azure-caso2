# azure-caso2
Este repositorio contiene un ejemplo de despliegue en Azure con terraform y la configuración con Ansible de Kunbernetes 
y la instalación de la aplicación de Jenkins.

En la carpeta terrafom está el código que ha referencia al despliegue de dos máquinas virtuales.
En la carpeta ansible se encuentran los diferentes play-book que se utilizan tanto para levantar Kubernes comopara instalar el docker que contiene Jenkis, en esta carpeta encontrareis dos carpetas una donde se encuentran los diferentes roles y variables que necesitamos para los diferente playbooks
